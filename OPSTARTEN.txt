OSINT Nieuwsdashboard opstarten
================================

1. Open Terminal
   Druk op Cmd + Spatie, typ "Terminal" en druk op Enter.

2. Navigeer naar de projectmap
   Typ het volgende en druk op Enter:
   cd "/Users/brink/Lokaal/loep"

3. Start de server
   Typ het volgende en druk op Enter:
   python3 -m http.server 8081

   Je ziet dan: Serving HTTP on 0.0.0.0 port 8081 ...

4. Open het dashboard in je browser
   Ga naar: http://localhost:8081

Stoppen: ga terug naar Terminal en druk op Ctrl + C.
